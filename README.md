# Forge — Landing Page

Landing page de [Forge](https://github.com/ronaldjdev/forge), un sistema operativo arquitectónico para backend.

## Stack

- React 19 + TypeScript 6
- Vite 8 + Tailwind CSS v4
- Oxlint

## Scripts

| Comando | Descripción |
|---------|-------------|
| `pnpm dev` | Servidor de desarrollo con HMR |
| `pnpm build` | Typecheck + build producción |
| `pnpm lint` | Lint con oxlint |
| `pnpm preview` | Previsualizar build |
| `pnpm deploy` | Build + publicar en GitHub Pages |
